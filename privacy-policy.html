Política de Privacidade - ClipCash

Última atualização: 21 de fevereiro de 2026

1. Introdução

A ClipCash ("nós", "nosso" ou "Empresa") opera a plataforma ClipCash ("Serviço"). Esta página informa você sobre nossas políticas sobre a coleta, uso e divulgação de dados pessoais quando você usa nosso Serviço e as opções que você tem associadas a esses dados.

2. Coleta de Dados

2.1 Dados que Você Fornece Diretamente

Quando você se registra e usa o Serviço, coletamos:

  • Informações de Conta: Nome, email, senha (criptografada), data de nascimento
  • Informações de Perfil: Foto de perfil, biografia, localização
  • Informações de Pagamento: Chave PIX (CPF, CNPJ, Email ou Telefone), dados bancários
  • Informações de Mídia Social: Nome de usuário, ID de conta, token de acesso
  • Conteúdo: Vídeos gerados, descrições, hashtags, metadados

2.2 Dados Coletados Automaticamente

Quando você usa o Serviço, coletamos automaticamente:

  • Dados de Dispositivo: Tipo de dispositivo, sistema operacional, identificadores únicos
  • Dados de Uso: Recursos usados, tempo gasto, cliques, navegação
  • Dados de Localização: Localização aproximada baseada em IP (se permitido)
  • Dados de Cookies: Identificadores de sessão, preferências de usuário
  • Logs do Servidor: Endereço IP, tipo de navegador, páginas visitadas

3. Uso de Dados

Usamos seus dados pessoais para:

  • Criar e manter sua conta
  • Processar pedidos de saque
  • Gerar vídeos e publicar em suas contas
  • Fornecer suporte ao cliente
  • Analisar uso do Serviço
  • Melhorar funcionalidades
  • Detectar e prevenir fraude
  • Cumprir obrigações legais

4. Compartilhamento de Dados

Compartilhamos seus dados com:

  • Provedores de Serviço: Processadores de pagamento, provedores de hospedagem, serviços de análise
  • Plataformas de Mídia Social: TikTok e Kwai (quando você publica vídeos)
  • Parceiros de Publicidade: Redes de CPA e afiliados
  • Conformidade Legal: Autoridades governamentais (se exigido por lei)

5. Retenção de Dados

Retemos seus dados pessoais enquanto você tiver uma conta ativa. Após o encerramento:

  • Dados de conta: Retidos por 1 ano para fins contábeis
  • Dados de transação: Retidos por 5 anos (conforme exigido por lei)
  • Dados de uso: Deletados após 90 dias
  • Dados de conteúdo: Deletados após solicitação

6. Segurança de Dados

Implementamos medidas técnicas e organizacionais para proteger seus dados:

  • Criptografia SSL/TLS para transmissão
  • Criptografia de senhas com bcrypt
  • Tokens de acesso armazenados com segurança
  • Firewalls e detecção de intrusão
  • Backups regulares

7. Seus Direitos

Você tem o direito de:

  • Acesso: Solicitar uma cópia de seus dados pessoais
  • Correção: Solicitar correção de dados imprecisos
  • Exclusão: Solicitar a exclusão de seus dados (direito ao esquecimento)
  • Portabilidade: Solicitar seus dados em formato estruturado
  • Restrição: Solicitar que limitemos o processamento
  • Objeção: Objetar ao processamento para fins específicos

Para exercer esses direitos, entre em contato conosco em: privacy@clipcash.app

8. Cookies

Usamos cookies para:

  • Manter você conectado
  • Lembrar suas preferências
  • Analisar uso do Serviço
  • Melhorar experiência do usuário

9. Conformidade com LGPD

Como empresa brasileira, estamos em conformidade com a Lei Geral de Proteção de Dados (LGPD). Você pode exercer seus direitos sob a LGPD conforme descrito na Seção 7.

10. Contato

Para dúvidas sobre esta Política de Privacidade, entre em contato conosco em:

  • Email: privacy@clipcash.app
  • Website: https://clipcash.app

11. Alterações à Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos você sobre alterações significativas por email ou através do Serviço.
